Each line consists of 5 fields separated by TAB or whitespace (similar to cron):

MINUTE (0-59)
HOUR (0-23)
DOM (1-31)
MONTH (1-12)
DOW The day of the week (0-7) where 0 and 7 are Sunday.

In addition:

  • * specifies all valid values
  • M-N specifies a range of values
  • M-N/X or */X steps by intervals of X through the specified range or whole valid range
  • A,B,...,Z enumerates multiple values

    • The H symbol can be used with a range. For example, 00 1 H(12-17) * 7

    Example: 00 11,16 * * * (11:00 and 16:00 everyday at 12:00am)
